# 🚀 Sistema de Cadastro de Usuários

Aplicação fullstack para gerenciamento de usuários, desenvolvida com React no frontend e Node.js com Express no backend, utilizando MongoDB como banco de dados.

---

## 🧑‍💻 Autor

Desenvolvido por Moisés Figueiredo

---

## 🧠 Sobre o projeto

Este projeto tem como objetivo demonstrar a construção de uma aplicação completa (fullstack), integrando frontend e backend em um fluxo funcional de cadastro e listagem de usuários.

A aplicação permite a comunicação entre uma interface desenvolvida em React e uma API REST construída com Node.js, responsável pelo processamento das requisições e persistência dos dados no banco MongoDB.

---

## 🎯 Objetivo

O projeto foi desenvolvido com foco em aprendizado e prática de conceitos essenciais do desenvolvimento web moderno, como:

* Criação de APIs REST

## 📡 Arquitetura da aplicação

A aplicação segue uma arquitetura cliente-servidor:

* O **frontend (React)** é responsável pela interface e envio de dados
* O **backend (Node.js + Express)** processa as requisições e define as regras de negócio
* O **MongoDB** armazena os dados dos usuários

A comunicação entre as camadas é feita através de requisições HTTP no padrão REST.
